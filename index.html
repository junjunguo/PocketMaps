Pocket Maps

Free offline maps with routing functions





Navigation

The application uses offline maps to a GPS navigation system to provide turn-by-turn guided instructions on how to arrive at a given destination.

Map

Use OpenStreetMap data, an open-source, free Wiki World Map. Can be downloaded by country or region.

No internet needed

Pocket Maps works completely offline. Once Maps are downloaded no internet connection needed to run the app. No more roaming charges when travelling abroad .

Free

Absolutely free, free application, free maps download, no in-app purchases.

Components

Use OpenStreetMap data, graphhopper & mapsforge open-source api.

Open Source

PocketMaps is open source software with MIT License, source code: GitHub

Tracking

Recording GPS tracks, draw a polyline of locations where the device has been on map; can save tracking log as GPX file, which may be used for geotag photos etc.

Tracking Analytics

Dynamic calculate and analyze your moving distance, max speed, average speed and time usage with both real time live textual summaries and line charts.

Sport Tracking

Measure your sport performance and calories burned dynamically. (see Tracking Analytics)

Always know where you are without network access, never get lost with your mobile GPS and Pocket Maps.
